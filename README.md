# 🎈 Bubble Game — Мини-игра на чистом JavaScript

Первый проект в рамках моего личного спринта по прокачке фронтенд-навыков. Это интерактивная игра, которая тренирует скорость и внимательность.

## 🚀 Демонстрация (Live Demo)
👉 [Посмотреть игру вживую](https://buble-game-1.netlify.app/) 

---

## 🕹️ Как играть:
1. Нажмите кнопку **Start** для запуска игры и таймера.
2. В верхней панели в поле **Hits** появится случайное число-цель.
3. Найдите на поле шарик с этим числом и кликните по нему.
4. За каждое правильное попадание вы получаете **+10 очков**, поле полностью обновляется, а цель меняется.
5. Игра продолжается, пока на таймере не закончится время (30 секунд).

---

## 🛠️ Технологии, которые я использовал:
* **HTML5** — семантическая структура игры.
* **CSS3** — адаптивная верстка, флексбоксы (`Flexbox`), анимация наведения (`hover`).
* **JavaScript (Vanilla JS)** — вся логика игры:
  * Генерация случайных чисел (`Math.random`).
  * Работа с таймером и интервалами (`setInterval`).
  * **Делегирование событий (Event Delegation)** — обработка кликов по 200+ шарикам через одного общего родителя ради высокой производительности.
